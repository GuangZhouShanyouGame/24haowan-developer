# 24haowan-developer

24好玩开发者知识共享库

详细请查看：[Gitbook](https://24haowan.gitbooks.io/developer/content/)

## 共享要求

### 1\. 内容

- 至少使用过或demo过，不允许直接转载。

### 2\. 目录

例子：IDE-Atom-packages.md

- 最多2层目录
- 形式：分类-名称-文档

### 3\. 文件命名

- 文件名要求为全小写英文，单词间以"-"连接。

### 4\. 更新

- 添加文档后需要更新索引：SUMMARY.md
- 添加一级目录`(如IDE/Atom)`后，需要在上级的文档`(如IDE/README.md)`中添加Links`(如Atom的官网地址)`
